# Understanding-Machine-Learning

A structured repository where I implement and explain one Machine Learning model every weekend (Saturday and Sunday).

This project focuses on building strong fundamentals through consistent practice, covering both theory and implementation of machine learning algorithms.

---

## Objective

- Develop a clear understanding of machine learning models
- Implement algorithms from scratch
- Build consistency through a fixed weekly schedule
- Create a well-organized public reference for ML concepts

---

## Learning Strategy

This repository follows a weekend-based execution plan:

- One model is added every Saturday and Sunday
- Each notebook includes:

  - Concept explanation
  - Mathematical intuition
  - Implementation from scratch
  - Visualization where applicable
